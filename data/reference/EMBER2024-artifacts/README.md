# EMBER2024 Official Artifacts

This directory contains the official EMBER2024 model artifacts that fit within
normal GitHub file-size constraints.

Tracked here:

- per-file-type detection models
- aggregate detection models
- behavior, exploit, group, packer, and file-property models

Not tracked here:

- `EMBER2024_family.model`
- multi-gigabyte memmaps
- large JSONL evaluation shards

Use `data/local/EMBER2024-artifacts` if you need the full local artifact set.
